# SyntaxWear - E-commerce de Tênis e Sneakers

Este projeto é um e-commerce fictício chamado **SyntaxWear**, especializado na venda de tênis e sneakers. O objetivo é apresentar um layout moderno, responsivo e organizado, focado em experiência do usuário e boas práticas de desenvolvimento front-end.

## Estrutura de Pastas

```
├── index.html                # Página principal do e-commerce
├── README.md                 # Documentação do projeto
├── css/                      # Arquivos de estilos CSS
│   ├── base.css              # Estilos base e utilitários globais
│   ├── layout.css            # Estilos de layout e grid
│   ├── reset.css             # Reset de estilos para cross-browser
│   ├── variables.css         # Variáveis CSS (cores, espaçamentos, etc)
│   └── components/           # Estilos de componentes específicos
│       ├── footer.css        # Footer do site
│       ├── header.css        # Header e navegação
│       ├── hero.css          # Seção principal (hero)
│       ├── product-category.css # Cards de categorias
│       └── product-grid.css  # Grid de produtos
├
├── images/                   # Imagens do projeto
│   ├── banners/              # hero.jpg, etc.
│   ├── favicons/             # Ícones de favoritos
│   ├── icons/                # Ícones SVG (bag, facebook, help, instagram, menu-toggle, tiktok, user, whatsapp)
│   ├── logo/                 # logo.svg
│   └── products/             # Imagens de produtos (card-imagem.jpg, casual.jpg, etc)
```

## Funcionalidades
- Layout responsivo e moderno
- Navegação por categorias (Masculino, Feminino, Outlet)
- Seção de destaque (hero)
- Grid de produtos e categorias
- Footer com navegação e redes sociais
- Formulário de inscrição para newsletter

## Tecnologias Utilizadas
- HTML5
- CSS3 (com organização modular por componentes)

## Como rodar o projeto
1. Clone este repositório
2. Abra o arquivo `index.html` em seu navegador

## Créditos e Observações
- Projeto desenvolvido para fins de estudo e portfólio.
- Imagens e marcas são fictícias.
